# Anonymous Feedback Web App

A **Next.js + MERN stack** web application that allows users to **send and receive anonymous feedback** with **AI-assisted message suggestions**, message toggling, and deletion functionality.

---

## Features

- **Anonymous Feedback**: Users can send messages without revealing their identity.
- **AI Message Suggestions**: The system suggests potential feedback messages to users.
- **Toggle Messages**: Users can **accept or reject** incoming messages.
- **Delete Messages**: Users can delete messages they donâ€™t want to keep.
- **Real-Time Updates**: Feedback is delivered instantly.
- **Modern Stack**: Built with MongoDB, Express, React, Node.js, and Next.js.

---

## Tech Stack

- **Frontend**: React, Next.js, TailwindCSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
-**AI Integration**: **Gemini AI model** for message suggestions
- **Real-Time Updates**: Socket.io
